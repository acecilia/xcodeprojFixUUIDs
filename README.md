# xcodeprojFixUUIDs

Make given .xcodeproj stable/keep merge conflicts in .xcodeproj to the minimum.

## Why

https://github.com/CocoaPods/CocoaPods/issues/9359

## How

```
mint run grigorye/xcodeprojFixUUIDs <your.xcodeproj>
```
